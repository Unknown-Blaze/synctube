# **App Name**: SyncTube

## Core Features:

- YouTube ID Input: Accepts a YouTube video ID from the user.
- Real-time Synchronization: Synchronizes playback across devices using Socket.IO.
- Sync Play Request: Requests the server to synchronize playback using a target timestamp.
- Sync Pause Request: Requests the server to pause the playback across all connected clients.
- Video Playback: Plays the YouTube video within the app.

## Style Guidelines:

- Background color: Dark background (#121212) for a modern, immersive experience.
- Primary color: Electric purple (#BF54F0) to represent synchronized energy.
- Accent color: Soft Lavender (#D0A2F7) for interactive elements, enhancing usability.
- Body and headline font: 'Inter' sans-serif, chosen for its modern, neutral appearance; excellent for both headlines and body text.
- Use minimalist icons for play, pause, and synchronization status.
- Centered video player with controls positioned at the bottom of the screen.
- Subtle loading animations during video synchronization.
